# ABC Analysis for Food Retail

## Task
ABC analysis by margin for September 2024 for the top sales store.

## Solution logic
1. Find store with highest September sales
2. Build ABC analysis by margin
3. Filter A-category SKUs with worse turnover vs August
4. Rank by 3-month profitability
5. Return top-5 SKUs

## Stack
- PostgreSQL
- SQL
- CTE
- Window functions
- BI analytics
